# IUCN Red List statistics

Retrieve count of species with assessments. This endpoint returns the
total number of assessed species on the IUCN Red List.

## Usage

``` r
rl_statistics()
```

## Value

A tibble (class `tbl_df`, `tbl`, `data.frame`) containing count and date
of access.

## Examples

``` r
if (FALSE) { # \dontrun{
# Get total count of assessed species
rl_statistics()
} # }
```
