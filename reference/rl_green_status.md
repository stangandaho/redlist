# Green status

Retrieve all IUCN Green Status assessments.

## Usage

``` r
rl_green_status()
```

## Value

A tibble (class `tbl_df`, `tbl`, `data.frame`) where each column
represents a unique API response JSON key. The columns include key
information about the Green Status assessment, such as year, weights,
justification, and other relevant metadata.
